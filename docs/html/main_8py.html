Detailed Description

This file contains a demonstration program that runs some tasks, an inter-task shared variable, and a queue.

The tasks don't really do anything; the example just shows how these elements are created and run.

This file contains the code to succesfully run 2 motors with motor control according to lab3 of ME-405.

  • the motors run a step response and dump that data to the terminal in which it can be copied to a csv (this can be replaced by a uart or other connection but is simple this way)
  • only the data from the step response is printed to the terminal once the rest of the data from the run is not printed to the terminal
